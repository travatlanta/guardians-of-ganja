# Client Logo Drop Zone

Place background-removed logo files here.

## Preferred format
- PNG with transparent background
- White or light logo marks preferred for dark background
- Suggested max canvas: 1200x600

## Naming
Use simple lowercase hyphen names, for example:
- client-01.png
- client-02.png
- client-03.png

Once added, these can be wired to the logo strip on the home page.
